# Minimal Pomodoro App Design Proposal

> 目标：设计一款桌面优先、之后可迁移到 iPhone 的极简番茄钟软件。界面文字全部使用英文，字体使用 Georgia。整体视觉为米白纸感，并加入一个低饱和强调色。

## 1. Inspiration Research

参考方向来自以下产品和可视化范式：

- Pomofocus: 可自定义番茄钟、短休息、长休息与任务结合，适合作为基础计时结构参考。  
  https://pomofocus.io/
- Forest: 将专注转化为可收藏、可回看的视觉资产，适合作为轻量奖励机制参考。  
  https://forestapp.cc/
- Flow: 极简、安静、专注的 Pomodoro 桌面体验，适合作为节奏与界面克制感参考。  
  https://www.flow.app/
- Focus To-Do: Pomodoro 与任务、统计结合，适合作为工作记录和报表参考。  
  https://www.focustodo.cn/
- TickTick: 任务管理与专注统计结合，适合作为任务-时间闭环参考。  
  https://ticktick.com/about/features
- Toggl Track: 时间轴、报表、时间记录思路，适合作为可视化结构参考。  
  https://toggl.com/track/
- Apple Activity Rings: 用极简环形图表达每日目标完成度，适合做 compact daily summary。  
  https://support.apple.com/en-us/105002
- GitHub contribution graph: 用日历热力图表达长期积累，适合做 monthly/yearly focus archive。  
  https://docs.github.com/

## 2. Product Positioning

这款软件不是复杂项目管理工具，而是一个安静、纸感、可长期使用的个人专注仪表盘。

核心价值：

- 用户可以自由设定 `Focus`, `Short Break`, `Long Break`, `Cycles`。
- 每天的工作不是只留下数字，而是留下直观、好看的视觉记录。
- 桌面端首先服务长时间工作场景，之后迁移到 iPhone 时保留相同的数据结构和视觉语言。
- 所有界面文本使用英文，避免中英文混排破坏 Georgia 的气质。

## 3. Visual Direction

### Typography

- Primary font: `Georgia`
- Fallback: `Georgia, Times New Roman, serif`
- UI text language: English only
- 字体风格：轻文学感、纸质感、不过度装饰。

### Color System

基础色建议：

- Paper Background: `#F7F0E6`
- Warm Surface: `#FFFBF3`
- Ink Text: `#28231D`
- Muted Text: `#837B70`
- Fine Line: `#DED3C3`

强调色待确认，推荐先以低饱和鼠尾草绿作为默认方向：

- Recommended Accent / Sage: `#8FA78A`
- Alternative Accent / Dusty Blue: `#8DA3B8`
- Alternative Accent / Clay Rose: `#B9897D`

判断说明：米白纸感需要一个冷静、不过甜的强调色。鼠尾草绿更像“专注与恢复”，蓝色更像“工具与秩序”，陶土玫瑰更像“手账与情绪记录”。最终强调色建议由你确认。

### Texture & Shape

- 背景可使用非常轻的纸纹噪点，透明度低，不能影响阅读。
- 卡片圆角不超过 `8px`。
- 不使用强渐变、大面积阴影、霓虹色、玻璃拟态。
- 图表线条使用细线与小面积填充，不做沉重 dashboard 风格。

## 4. Core Timer Experience

桌面主界面应直接进入可用状态，而不是 landing page。

主界面模块：

- `Today`
- `Review`
- `Settings`

主计时器区域：

- Large timer: `25:00`
- Current state: `Focus`, `Short Break`, `Long Break`
- Cycle indicator: `Cycle 2 of 4`
- Main actions: `Start`, `Pause`, `Resume`, `Skip`, `Reset`
- Optional action: `End Day`

可配置项：

- `Focus Length`
- `Short Break`
- `Long Break`
- `Cycles`
- `Auto-start Breaks`
- `Auto-start Focus`
- `Sound`
- `Desktop Notification`

建议预设：

- `Classic`: 25 / 5 / 15 / 4
- `Deep Work`: 50 / 10 / 30 / 3
- `Writing`: 40 / 8 / 20 / 4
- `Reading`: 30 / 5 / 15 / 4

## 5. Visualization Concept A: Geometric Calendar

工作名：`Focus Quilt`

这是一个极简几何日历/热力格子视图，用来展示一段时间内每天的专注积累。它可以借鉴 GitHub contribution graph 的一眼扫描能力，但视觉上更像纸面上的小色块拼布。

### Visual Form

- 每一天是一个小方格或细长矩形。
- 完成的番茄钟越多，格子的填充越深。
- 当天若有完整周期，可以在格子角落出现一个很小的点。
- 不同工作标签可以用细线、纹理或小角标表示，而不是使用多种强色。

### What It Shows

- `Focused Minutes`
- `Completed Sessions`
- `Completed Cycles`
- `Longest Streak`
- `Best Day`

### Interaction

- Hover 某一天显示小浮层：`Mon, 24 Feb`, `145 min`, `6 sessions`, `2 cycles`
- 点击某一天进入该天详情。
- 月视图和年视图都可以使用这个系统。

### Strength

它最适合表达“长期坚持”。用户可以很快看出哪几天专注多、哪几天断掉、哪一周状态最好。

### Risk

它不太能表达一天内部的节奏，比如上午是否被打断、晚上是否连续工作。因此它适合作为 `Review` 的主视图或 `Today` 的小型辅助视图。

### Desktop Placement

- `Review` 页面主视图。
- `Today` 页面右侧放一个本周 mini quilt。

### iPhone Migration

- 采用纵向滚动的月历。
- 每个月作为一个 section。
- 点击日期后从底部弹出 day detail。

## 6. Visualization Concept B: Time River

工作名：`Day Thread`

这是一天内部的时间轴/时间河流，用来表达今天的工作节奏。它比传统列表更直观，可以看到专注、休息、空白、打断如何分布。

### Visual Form

- 横向时间轴从 `06:00` 到 `24:00`。
- `Focus` 是较深的细长色带。
- `Break` 是浅色短带。
- 空白时间保留纸色。
- 被中断的 session 用断开的虚线或缺口表示。
- 当前进行中的 session 轻微呼吸动画，但可以关闭。

### What It Shows

- 工作集中在一天的哪个时段。
- 是否频繁中断。
- 休息是否规律。
- 今日还剩多少目标时间。

### Interaction

- Hover 某段显示：`Focus`, `09:30-09:55`, `25 min`
- 点击某段可以添加短 note。
- 当前 session 自动在时间轴上生长。

### Strength

它最适合作为桌面端 `Today` 页面核心可视化。用户不需要读报表，只看一条线就能理解今天的节奏。

### Risk

如果一天内记录过多，横向时间轴可能变密。解决方式是只显示番茄钟段落，不显示每一分钟细节；详情放到 hover 或点击后展示。

### Desktop Placement

- `Today` 页面计时器下方。
- 宽屏桌面端可以展示完整横向时间轴。

### iPhone Migration

- 改为纵向 timeline。
- 每个 session 作为一段细卡片，但仍保持纸感和细线样式。
- 当前 session 固定在顶部，历史向下滚动。

## 7. Visualization Concept C: Growth Collection

工作名：`Focus Herbarium`

这是一个轻量收藏式可视化。每完成一个番茄钟或一个完整周期，用户会获得一个极简纸面标记，例如 pressed leaf、stamp、small tile。它借鉴 Forest 的“专注产生可回看成果”的思路，但不走卡通路线，而是更安静、更手账化。

### Visual Form

- 每个完成的 `Focus` 生成一个小叶片、纸印章或几何贴片。
- 一个完整 `Cycle` 生成一个更完整的组合图案。
- 一天结束后，这些标记排列成一个小型 collection。
- 视觉上像 paper herbarium 或 stamp book。

### What It Shows

- 今天完成了多少个 session。
- 哪些 session 是连续完成的。
- 哪些 session 被中断。
- 哪一天形成了特别完整的 collection。

### Interaction

- 完成 session 时出现一个非常轻的生成动画。
- `End Day` 后出现当天 collection summary。
- 点击 collection item 可以看到当时的时间、标签与 note。

### Strength

它最适合增加“有趣”和“可纪念性”。相比单纯数字，它会让每天的工作成果更有存在感。

### Risk

如果做得太可爱，会偏离极简目标。建议采用植物标本、纸张印章、几何贴片这类克制隐喻，而不是游戏化角色。

### Desktop Placement

- `Today` 页面底部作为当天成果区。
- `Review` 页面点击某一天后显示该天 collection。

### iPhone Migration

- 适合做成 swipeable daily cards。
- 每张 day card 展示当日 collection 与关键数字。
- 可以作为移动端最具情绪价值的回顾入口。

## 8. Recommended Screen Layout

### Desktop: Today

左侧：

- `Today`
- `Review`
- `Settings`

中间：

- Large timer
- Timer controls
- `Day Thread`

右侧：

- `Focus Quilt` mini week/month preview
- `Focus Herbarium` today collection
- Quick settings: `Focus Length`, `Cycles`

### Desktop: Review

顶部：

- `This Week`, `This Month`, `This Year`

主体：

- `Focus Quilt` as main calendar view

侧栏或下方：

- Selected day detail
- `Day Thread`
- `Focus Herbarium`
- Key numbers: `Focused Minutes`, `Sessions`, `Cycles`, `Streak`

### Desktop: Settings

设置保持简短：

- `Timer`
- `Appearance`
- `Notifications`
- `Data`

## 9. iPhone Migration Strategy

桌面端先做宽屏效率，移动端保留核心体验但重新布局。

迁移原则：

- 不直接压缩桌面三栏，而是改为底部 tab。
- `Today` 放计时器与当前 session。
- `Review` 放 calendar 和 day cards。
- `Settings` 放周期、声音、通知。
- `Day Thread` 从横向变纵向。
- `Focus Quilt` 从桌面宽屏变为月度滚动。
- `Focus Herbarium` 成为移动端回顾的视觉亮点。

移动端底部 tabs：

- `Today`
- `Review`
- `Settings`

## 11. Motion & Feedback

动画必须非常轻：

- 计时器进度可以用细环或细线。
- `Day Thread` 当前段落可缓慢增长。
- 完成 session 时，`Focus Herbarium` 添加一个小标记。
- 支持 `Reduce Motion`，关闭所有非必要动画。

声音建议：

- `Soft Bell`
- `Paper Tap`
- `None`

## 12. Accessibility

- 所有信息不能只依赖颜色，还要通过深浅、纹理或数字表达。
- Georgia 字体在小尺寸下可读性有限，小字建议不低于 `13px`。
- 主计时器数字可以大，但不使用过细字重。
- 所有按钮支持键盘操作。
- 颜色对比度需要在浅纸色背景上验证。

## 13. MVP Scope

第一版建议只做真正有用的闭环：

- 自定义番茄钟时长与周期。
- 开始、暂停、跳过、重置。
- 本地保存 session。
- `Day Thread` 今日时间轴。
- `Focus Quilt` 月度回顾。
- `Focus Herbarium` 今日成果。
- 英文 UI + Georgia + 米白纸感主题。

第二版：

- 标签系统。
- 周报/月报导出。
- iCloud 或账号同步。
- iPhone 版本。
- 更多 accent color。

## 15. Open Design Decisions

这些是建议你下一步确认的设计点：

1. 强调色选择：`Sage`, `Dusty Blue`, or `Clay Rose`。
2. `Focus Herbarium` 的隐喻：pressed leaf、paper stamp、还是 geometric tile。
3. 是否需要任务列表。如果需要，建议只做轻量 `Today Tasks`，不要变成完整 todo app。
4. 是否需要登录与云同步。桌面 MVP 可以先本地存储。
5. 提醒声音是否默认开启。极简产品建议默认关闭或使用非常轻的 `Soft Bell`。
